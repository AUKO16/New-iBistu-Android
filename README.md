# iBistu安卓版

#### 作者：徐鼎

---
## 项目简介
北京信息科技大学校园APP，具备以下功能：

1. 及时了解校内最新资讯新闻；
2. 随时查阅校内黄页；
3. 校区地图；
4. 校内二手交易；
5. 校内全景图；
6. 校园班车查询；
